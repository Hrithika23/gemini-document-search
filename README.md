A simple chatbot using gemini model to generate answer to a particular question from the uploaded pdf using the keyword search.

Keyword search(1):
steps:
1)Gemini API setup.
2)Pdf text extraction using PyPDF2 reader.
3)Text chunking 
4)Keyword search for relevant question.
5)Gemini responses using the model, prompts and config provided.
6)Streamlit UI
-uploading the pdf file 
-Ask any questions from pdf 
-Generates the relevant answers from the pdf file 

keyword search2(2)
steps:1-5 same as above steps 
6)The pdf need to be uploaded in the console and the question need to be asked in console and the output will be generated .
